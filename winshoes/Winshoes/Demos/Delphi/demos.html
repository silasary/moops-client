Demos are in Kudzu\Winshoes\Demos\ on your local drive

This may not be a complete list of demos, check your hard drive for a complete list.

BFtp

Backup FTP - This is a demo which performs backups via FTP.

CGI_Mailer

CGI Mailer is a CGI program which takes form input and EMails the results to a specified Email address. This is the application that took your information before you downloaded winshoes. This demo requires Web Broker, which is in the Client / Server edition or can be purchased seperately.

Chat

Chat client and server.

CS

Simple Client / Server demo using TCP/IP.

FTP_Put_File

Command line program for FTPing files. Great for batch files! This program is used in conjunction with a batch file to automate the uploads of the winshoes install to the server.

HTTP_Server

A very simple web server. This demo still under construction.

Mail

A very simple program to send EMail.

MailDLL

This is an example of a DLL which encapselates the SMTP component to send mail. This DLL can be used from any langauge.

NGScan

This demo is still under construction. Newsgroup scan will produce load data from a new server into a paradox database, and produce statistic reports.
